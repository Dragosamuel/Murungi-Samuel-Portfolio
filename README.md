# Master Guide Murungi Samuel - Portfolio Website

This is a professional portfolio website for Master Guide Murungi Samuel, a Senior Youth Leader in the Seventh-day Adventist Church.

## Features

- Responsive design that works on all devices
- Blue, white, and gold color scheme representing the Adventist colors
- Interactive elements with smooth animations
- Gallery section with image lightbox
- Contact form
- Downloadable documents section
- Testimonials from colleagues and church leaders

## Technologies Used

- HTML5
- CSS3 with Bootstrap 5
- JavaScript
- Font Awesome Icons

## File Structure

```
├── index.html          # Main HTML file
├── styles.css          # Custom CSS styling
├── script.js           # JavaScript functionality
├── favicon.ico         # Website favicon
├── profile.jpg         # Profile image
├── profile1.jpg        # Background image for hero section
├── camp1.jpg           # Ministry activity image
```

## How to View the Website

1. Simply open `index.html` in any modern web browser
2. The website is fully responsive and will adapt to mobile, tablet, and desktop screens

## Customization

To customize the content:

1. Edit `index.html` to update personal information, testimonials, and content
2. Modify `styles.css` to change colors, fonts, or spacing
3. Update `script.js` to modify interactive behaviors
4. Replace images with your own photos

## Sections

1. **Hero Section** - Introduction with profile picture
2. **About Me** - Personal biography and spiritual journey
3. **Ministry Journey** - Overview of ministry experiences
4. **Leadership Development** - Leadership skills and training
5. **Community Service** - Outreach and service activities
6. **Skills & Honors** - Professional skills and achievements
7. **Gallery** - Photo gallery of ministry activities
8. **Testimonials** - Recommendations from colleagues
9. **Contact** - Contact information and form
